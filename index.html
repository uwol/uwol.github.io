Projects

ANTLR4 grammar for COBOL 85

2015 - Present
uwol/cobol85grammar

Authorship of the ANTLR4 grammar for the programming language COBOL 85 in the ANTLR4 open source project.

The grammar passes the NIST COBOL test suite and has been applied for parsing COBOL production source code.

ANTLR4 grammar for Visual Basic 6.0

2014
uwol/vb6grammar

Authorship of the ANTLR4 grammar for the programming language Visual Basic 6.0 in the ANTLR4 open source project.

A derivative of the grammar is applied at the core of the Rubberduck VBA project, which enables unit testing, refactoring and code inspection in the Visual Basic Editor (VBE).

Agent-based computational economy

2013 - 2015
uwol/ComputationalEconomy

Development of an agent-based computational economics simulator, which implements an Arrow-Debreu-like general equilibrium model according to neoclassical theory and traditional marginalism based on agent types household, factory, trader, credit bank, central bank and state.

Presented at 21st Int. Conf. on Computing in Economics and Finance (CEF 2015), June 20-22, 2015, Taipei, Taiwan.
